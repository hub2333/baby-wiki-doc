# AAP与CDC-虫咬蜱虫与蜂蜇边界

## 来源摘要

这组来源主要支撑 0-3 岁家庭最常见的几类场景：普通虫咬肿包到底算不算严重、蜂蜇后什么时候要警惕全身过敏、蜱虫应该怎么取、哪些后续症状值得尽快联系医生。

## 可直接支撑的结论

- AAP 明确指出，多数普通虫咬和蚊虫包是局部反应，红肿在头 2-3 天内变大一点仍可见于正常反应，不等于感染
- 蜂或黄夹克蜇伤后，局部红、痛、痒、肿是常见反应；真正危险的是全身过敏信号，如呼吸困难、吞咽困难、全身荨麻疹和进行性面唇肿胀
- AAP 的过敏页强调：已知严重蜂蜇过敏的孩子，一旦再蜇伤，哪怕一开始症状轻，也应尽快使用肾上腺素并呼叫急救
- AAP 与 CDC 都强调：蜱虫应尽快用尖头镊子贴近皮肤稳稳向上拔出，不要扭转、不要用凡士林、不要用火
- CDC 与 AAP 都提示：蜱虫咬后后续几天到几周若出现发热、皮疹或状态变差，需要进一步评估
- AAP 还提示：蜱虫附着时间较长、看起来胀大，或疑似鹿蜱高风险暴露时，需要和医生讨论是否需要进一步处理

## 适用限制

- AAP 与 CDC 可支撑普通虫咬、蜂蜇和蜱虫处理的家庭分流，但不能替代医生判断是否需要抗过敏急救药、抗生素或蜱虫暴露后预防。
- 蜱媒疾病风险有明显地区差异；是否需要预防性用药不能只靠家庭观察蜱虫外观决定。
- 已知严重蜂蜇过敏的孩子属于特殊高风险人群，应按医生预案处理，本知识库不提供肾上腺素剂量或处方建议。

## 与其他来源的关系

- 与 [[95-来源笔记/AAP与MedlinePlus-宠物咬伤抓伤与人咬伤边界]] 区分：虫咬蜂蜇重点是局部反应、过敏和蜱媒病；咬伤抓伤重点是伤口污染、狂犬病和深部损伤。
- 与 [[95-来源笔记/AAP-门诊急诊与120分流]] 一致：呼吸困难、吞咽困难、脸唇舌肿、反应差属于立即急救信号。
- 与皮肤护理来源互补：普通虫咬局部红肿可先按皮肤护理处理，但越来越痛、发热、流脓或全身症状要升级。

## 页面级映射

- [[20-主题知识/虫咬、蜱虫与蜂蜇边界]]
  - 支撑普通虫咬红肿 1-3 天内变大一点不一定等于感染。
  - 支撑蜂蜇后重点观察呼吸、吞咽、声音、脸唇舌肿、荨麻疹、呕吐和精神状态。
  - 支撑蜱虫用尖头镊子贴近皮肤稳定向上拔，不扭、不涂油、不用火。
  - 支撑蜱虫咬后几天到几周出现发热、红环样皮疹或状态变差需就医。
