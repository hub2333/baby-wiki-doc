# AAP与NIDDK与MedlinePlus-尿路感染尿痛与发热边界

## 来源摘要

这组来源主要支撑 0-3 岁家庭在几类高频场景下的判断：小婴儿只有发热时要不要想到 UTI、尿痛和尿频什么时候该验尿、哪些信号更像感染波及到肾脏、以及便秘和局部刺激为什么会和 UTI 混在一起。

## 可直接支撑的结论

- AAP 明确写到：**`2 岁以下` 婴幼儿发生 UTI 时，可能几乎没有典型症状，除了发热很难看出来**
- AAP 还提示：在婴儿里，**没有明显呼吸道感染或腹泻解释的高热**，可能就是尿路感染的唯一线索
- NIDDK 指出：`2 岁以下` 孩子的膀胱感染症状可包括发热、烦躁、吃奶差、体重增长差、呕吐或腹泻；尿检是确认诊断的关键
- AAP 和 MedlinePlus 都提示：较大一些的孩子如果有 **尿痛、尿频、尿急、尿血、尿臭、突然再次尿裤子**，都应尽快联系医生评估
- NIDDK 指出：**发热、寒战、恶心呕吐、背侧/腰侧痛** 更要警惕感染已经上行到肾脏
- AAP 指出：**症状本身不够诊断 UTI**，尤其小月龄孩子通常需要规范采集尿样；对未如厕训练的婴幼儿，若初筛可疑，往往需要导尿标本来确认
- AAP 还提醒：**便秘** 和局部刺激可带来与 UTI 相似的症状；预防上要避免憋尿、减少刺激性泡泡浴和香精清洁用品

## 适用限制

- 这些来源支持“想到 UTI 并尽快验尿”，但不能用症状直接确诊，也不能替代尿检、尿培养和医生判断。
- 抗生素选择、疗程、是否需要影像检查与年龄、发热程度、尿培养结果和既往病史有关；本知识库不提供自行用药建议。
- 小于 3 个月发热属于更高风险场景，应按发热分流优先处理，不只按尿痛页面观察。

## 与其他来源的关系

- 与 [[95-来源笔记/AAP-发热与婴儿发热]] 互补：小婴儿不明原因发热需要想到 UTI，但发热红旗仍按总分流处理。
- 与 [[95-来源笔记/AAP-便秘与排便问题]] 互补：便秘可加重尿频、憋尿和 UTI 风险，也可能模拟尿路症状。
- 与外阴/尿布区刺激来源区分：尿痛可能来自局部刺激，但合并发热、精神差、呕吐或腰侧痛时不能只按皮肤刺激处理。

## 页面级映射

- [[20-主题知识/尿路感染、尿痛与发热边界]]
  - 支撑 2 岁以下 UTI 可能只有发热、烦躁、吃奶差、呕吐等非特异表现。
  - 支撑尿痛、尿频、尿急、尿血、尿臭、突然尿裤子需要尽快联系医生或验尿。
  - 支撑发热合并腰侧/背部痛、反复呕吐、精神差时警惕上行感染。
  - 支撑不要自行先用剩余抗生素，以免影响尿检和尿培养判断。
