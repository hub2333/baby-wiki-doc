# PoisonHelp与AAP-中毒误服和窒息

## 来源摘要

这组来源对应家庭里最让人慌乱的两种高风险情境：孩子误服有毒物质，以及孩子发生食物或小物件窒息。它们共同强调：**先识别是否已进入急症，再决定是立即急救还是马上联系中毒/医疗专业支持。**

## 可直接支撑的结论

- 怀疑中毒时不要自行催吐
- 如果孩子已经出现呼吸困难、抽搐、倒下或叫不醒，应立即呼叫急救
- 美国官方毒物支持热线是 1-800-222-1222；在美国以外，应尽快联系当地急救系统或毒物咨询机构
- 0-3 岁儿童常见窒息来源包括整粒坚果、整粒葡萄、硬糖、爆米花、气球、小电池、小磁铁和细小玩具件
- 高致敏食物引入时要同时考虑过敏风险和窒息风险

## 适用限制

- Poison Help 和 Poison Control 的电话与部分处置流程基于美国系统；中国或其他地区应替换为当地急救、急诊或毒物咨询渠道。
- 中毒与窒息是两类不同急事：中毒重点是毒物暴露和专业咨询，窒息重点是气道是否堵塞和现场急救。
- 本知识库只做风险分流，不替代 CPR / 海姆立克 / 婴幼儿急救培训。
- 纽扣电池和磁珠有独立高风险页面，不能只按普通误服处理。

## 与其他来源的关系

- 与 [[95-来源笔记/AAP与PoisonControl-误吞纽扣电池与磁珠边界]] 互补：纽扣电池和多个磁珠按急症处理，即使早期症状轻也不能等待。
- 与 [[95-来源笔记/AAP-门诊急诊与120分流]] 一致：呼吸困难、抽搐、倒下、叫不醒优先呼叫急救。
- 与食物过敏来源区分：过敏红旗是全身过敏反应；窒息红旗是不能有效咳嗽、说不出/哭不出、发青或无力。
