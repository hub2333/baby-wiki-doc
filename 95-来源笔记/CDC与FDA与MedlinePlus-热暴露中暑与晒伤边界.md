# CDC与FDA与MedlinePlus-热暴露中暑与晒伤边界

## 来源摘要

这组来源主要解决 0-3 岁家庭在夏季最常见的几类判断问题：什么算普通晒伤或热不适、哪些表现已经更像热衰竭或热射病、6 个月以下婴儿如何做防晒、以及车内高温为什么必须按急症风险看待。

## 可直接支撑的结论

- MedlinePlus 指出，儿童属于更容易更早受到高温影响的人群；热相关急症本质上来自极端高温或日晒暴露
- MedlinePlus 将热相关表现区分为早期症状、热衰竭和热射病：热衰竭可见大量出汗、口渴、深色尿、头晕、头痛、恶心呕吐、无力、呼吸快；热射病则要警惕高体温、意识改变、抽搐和失去反应
- MedlinePlus 给出的现场处理重点是：转移到凉处、凉湿降温、清醒时少量补液；并明确提醒不要指望退烧药解决热相关急症
- MedlinePlus 还明确提醒：夏季绝不能把孩子单独留在被太阳暴晒的车里
- CDC 的日晒防护页强调：阴凉、遮挡衣物、帽子、墨镜和广谱防晒霜应配合使用；防晒霜需要补涂，不能只涂一次
- CDC 与 FDA 都明确写到：**6 个月及以下婴儿不建议常规使用防晒霜作为首选防护**，应优先避免正午暴晒并使用保护性衣物；6 个月以上儿童可按说明使用防晒霜
- FDA 说明：防晒霜应选择广谱产品，`SPF 15` 或更高；在外超过 `2` 小时，以及游泳、出汗或擦干后需要再次涂抹

## 适用限制

- CDC / FDA 的防晒建议主要面向美国公众，防晒产品标准、标签和 SPF 规则以美国监管语境为主。
- MedlinePlus 的热急症资料适合识别热衰竭和热射病红旗，但不替代急诊对脱水、电解质紊乱、感染或其他病因的鉴别。
- 0-3 岁儿童散热和补液能力弱；本知识库采用比成人更保守的分流表达。

## 与其他来源的关系

- 与 [[95-来源笔记/AAP-发热与婴儿发热]] 区分：热暴露后的高体温不能简单等同感染发热，也不能只靠退热药处理。
- 与 [[95-来源笔记/AAP-门诊急诊与120分流]] 一致：意识异常、抽搐、叫不醒、呼吸费力和严重脱水优先急救。
- 与皮肤护理来源互补：晒伤属于皮肤损伤，但合并发热、寒战、精神差或大水疱时需要升级分流。
