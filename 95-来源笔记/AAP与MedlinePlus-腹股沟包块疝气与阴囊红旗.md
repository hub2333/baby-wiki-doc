# AAP与MedlinePlus-腹股沟包块疝气与阴囊红旗

## 来源摘要

这组来源主要支撑家长对几类高频又容易误判的场景做分流：腹股沟或阴囊鼓包是不是疝气、阴囊肿大是不是鞘膜积液、什么时候可以尽快门诊、什么时候要按嵌顿疝或睾丸扭转的急症思路处理。

## 可直接支撑的结论

- AAP 指出，腹股沟疝常表现为腹股沟、阴囊，或女孩大阴唇附近的鼓包；鼓包通常会来回出现，哭闹、用力时更明显，放松后可变小
- AAP 明确提醒：**不要忽视婴幼儿任何腹股沟或阴囊包块**；嵌顿疝会表现为包块一直卡住不回去，并可伴疼痛、压痛、呕吐，包块还可能变硬、发红
- AAP 还说明：如果疝内容物血供受影响，可能出现更重的红旗，包括严重疼痛、呕吐、食欲下降、局部发红或淤青，甚至发热和便血
- AAP 关于鞘膜积液的文章说明：很多新生儿和婴儿会有鞘膜积液，通常表现为不太疼的阴囊肿大，很多会在约 1 岁前自行缓解
- AAP 也提示：如果出现**突然明显阴囊痛和阴囊肿胀**，要警惕睾丸扭转或肠管卡住，需立刻急诊评估
- MedlinePlus 指出：阴囊肿大或疼痛都值得尽快找医生判断；其中 **睾丸扭转是急症**，血供在数小时内受损就可能造成不可逆伤害
- MedlinePlus 还提示：鞘膜积液通常是无痛、像水球样的阴囊肿大；如果大小会随着腹部或阴囊受压而变化，更要考虑与腹股沟疝相通

## 适用限制

- 这些来源帮助家庭识别腹股沟/阴囊包块的红旗，但不能替代医生查体、超声或外科/泌尿专科判断。
- 鞘膜积液是否观察、何时手术与年龄、大小变化、是否交通性相关；不能仅靠“不疼”长期自行观察。
- 睾丸扭转是时间敏感急症；本知识库不提供家庭排除扭转的方法。

## 与其他来源的关系

- 与 [[95-来源笔记/AAP-门诊急诊与120分流]] 一致：剧烈疼痛、反复呕吐、反应差、无法安全转运时优先急救。
- 与 [[95-来源笔记/AAP与NIDDK与MedlinePlus-尿路感染尿痛与发热边界]] 区分：阴囊/腹股沟疼痛不应只按尿路感染或普通腹痛解释。
- 与腹泻呕吐来源互补：呕吐若合并腹股沟包块不回纳，要先警惕嵌顿疝。
