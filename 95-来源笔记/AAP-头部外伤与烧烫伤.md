# AAP-头部外伤与烧烫伤

## 来源摘要

这组来源适合把家庭里高频的跌落撞头、头部外伤和烧烫伤，整理成“先处理什么、哪些信号要尽快去急诊”的决策页。

## 可直接支撑的结论

- 头部受伤后，如果孩子反复呕吐、越来越困、意识异常、走路或说话异常、抽搐，应尽快急诊
- 婴幼儿不会表达头痛，持续大哭、难安抚、拒奶拒食、异常嗜睡也要提高警惕
- 烧烫伤后应立即用流动凉水冷却，不要用冰，不要自行刺破水疱
- 面部、手部、会阴、大面积、深度可疑或电击/化学烧伤，应尽快就医

## 适用限制

- AAP 头部外伤和烧伤页面适合做家庭分流，不用于判断是否一定存在脑震荡、颅内出血或烧伤深度。
- 婴幼儿外伤判断更依赖整体状态变化；“看起来还小、不会说痛”不能作为低风险依据。
- 烧伤面积、深度和部位需要医生判断；本知识库不提供家庭计算烧伤面积或自行处理深烧伤的方法。

## 与其他来源的关系

- 与 [[95-来源笔记/AAP-门诊急诊与120分流]] 一致：意识异常、抽搐、大出血、严重烧伤或状态差时优先急救。
- 与 [[95-来源笔记/AAP与MedlinePlus-割伤夹伤与出血处理边界]] 互补：外伤先看出血、功能、深度和污染风险。
- 与眼外伤、牙齿外伤来源互补：头面部外伤需要按受伤部位跳转到更细页面。

## 页面级映射

- [[20-主题知识/跌落头部外伤与烧烫伤]]
  - 支撑“越来越困、越来越吐、越来越不对劲”作为头部外伤后核心观察。
  - 支撑头外伤后反复呕吐、嗜睡、行为异常、抽搐、失去意识、拒奶拒食等就医信号。
  - 支撑烧烫伤后流动凉水冷却、不用冰、不涂偏方、不刺破水疱和覆盖干净敷料。
  - 支撑面部、手部、会阴、面积大、深度可疑、电击伤、化学烧伤等就医边界。
- [[20-主题知识/安全与伤害预防]]
  - 支撑高处防跌落、热饮热锅和洗澡水烫伤预防。
