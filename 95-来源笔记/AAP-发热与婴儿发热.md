# AAP-发热与婴儿发热

## 来源摘要

AAP / HealthyChildren 的发热相关文章适合作为 0-3 岁家庭判断“什么时候该观察，什么时候该尽快就医”的基础参考。

## 可直接支撑的结论

- 小于 3 个月婴儿体温达到或超过 38.0°C，应尽快就医
- 发热本身不是唯一判断标准，更要看精神状态、喝水吃奶情况、呼吸、皮疹和抽搐等伴随表现
- 任何年龄出现叫不醒、抽搐、呼吸困难、皮肤发紫、严重脱水等情况，都应尽快急诊评估

## 适用限制

- 来源面向家长，不用于区分具体感染类型，也不替代医生对败血症、脑膜炎、尿路感染等风险的判断。
- 体温阈值尤其适用于小于 3 个月婴儿；更大儿童不能只按温度决定是否就医。
- 退热药信息只作为舒适护理背景，本知识库不从该来源推导个体化剂量。

## 与其他来源的关系

- 与 [[95-来源笔记/AAP与CDC-热性惊厥与发热抽搐边界]] 一致：退热处理主要改善舒适度，不能保证预防热性惊厥。
- 与 [[95-来源笔记/AAP-门诊急诊与120分流]] 一致：呼吸、意识、抽搐、发青和严重脱水优先决定分流级别。
- 与尿路感染、皮疹、手足口等来源的关系：发热只是入口信号，伴随症状决定需要跳转到哪个专题页。
