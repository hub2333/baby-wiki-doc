# 优化清单

本页记录全盘审计后的后续优化方向，用来承接 [[99-项目管理/建设日志]] 里的下一步任务。

## 当前健康度

- Markdown 总数：160
- Obsidian 双链：无断链、无孤儿页
- frontmatter：所有 Markdown 都包含 `title`、`status`、`updated`
- 行尾：Markdown 与配置文件无 CRLF
- 主题页：均包含参考来源
- 分支：当前整理成果已推送到 `codex/baby-wiki-doc-progress`

## 已经比较稳的部分

- 0-3 岁主题页、高风险边界页和急救分流页已经形成较完整闭环。
- 高风险医疗/急救来源笔记已补充适用限制、来源关系、页面级映射和待更新项。
- 3-12 岁已经有年龄首页、阶段页、横向阅读路径和一批主题页。
- 模板已建立：[[99-项目管理/主题页模板]]、[[99-项目管理/来源笔记模板]]。

## 优先优化项

### 1. 继续补 3-12 岁主题页

优先把这些页面从首版摘要推进为带年龄分层和跨主题跳转的版本：

- [[20-主题知识/学校节律、睡眠与注意力（3-12岁）]]
- [[20-主题知识/常见感染与就医判断（3-12岁）]]
- [[20-主题知识/饮食挑食与体重管理（3-12岁）]]
- [[20-主题知识/运动、视力与姿势（3-12岁）]]
- [[20-主题知识/返校、缺勤与在家休息判断（3-12岁）]]
- [[20-主题知识/青春前期身体变化准备（9-12岁）]]

### 2. 提升 3-12 岁来源笔记

当前高风险急症来源笔记较完整，但多数 3-12 岁来源笔记仍偏摘要。优先补：

- 适用限制
- 与其他来源的关系
- 页面级映射
- 待更新项

### 3. 补中国本地来源

优先补中国本地公开来源，用来增强本地化表达：

- 儿童健康管理、学校卫生、近视防控、口腔保健
- 免疫规划、常见感染、返校和缺勤判断
- 急救、120、毒物咨询、狂犬病暴露处置

### 4. 建立定期体检脚本

当前检查依赖临时命令，后续可以沉淀为脚本：

- frontmatter 检查
- Obsidian 双链断链 / 孤儿页检查
- CRLF 检查
- 主题页是否包含参考来源
- 来源笔记是否包含质量标准字段

## 暂不优先

- 不急着引入复杂自动化或数据库化图谱。
- 不急着把所有 `draft` 改成 `active`，先保持内容仍可迭代。
- 不急着大规模重写 0-3 岁普通主题页，优先补 3-12 岁和来源层。
